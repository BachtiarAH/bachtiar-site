## The Details

Nulis Aksara Jawa is an educational purpose mobile app intend to teach student how to read, spel, and write javanese script using phone. It have automatic correction system powered YOLO v8 algorithm.

## The Features

- List of Javanese Script
- Sound how to pronounce the script
- Example how to use it
- training how to write with hand.
- Excercise to test reading and writing

This app are demo how object detection like YOLO v8 able to solf education problem how student hard to learn javanese script, you can read the research about how we design the solution for the problem here [Implementasi Yolo v8 Pada Aplikasi Edukasi Aksara Jawa](https://sipora.polije.ac.id/41643/)
